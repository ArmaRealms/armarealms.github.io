# Leilão Virtual de Jogadores

**Quantidade de itens no leião:**
| Cargo    | Quantidade de Itens no Leilão |
| :------: | :---------------------------: |
| Membro | 02 itens |
| Premium | 05 itens |
| Master | 10 itens |
| Ultimate | 15 itens |

**Tempo do item em leilão:**
| Cargo    | Tempo do Item no Leilão |
| :------: | :---------------------: |
| Membro | 02 dias |
| Premium | 04 dias |
| Master | 07 dias |
| Ultimate | 10 dias |

## Comandos do Leilão
| Comando | Descrição |
| ------- | --------- |
| `/ah` ou `/leilao` | Abrir o menu principal de leilão |
