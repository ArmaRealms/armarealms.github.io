# Introdução

Aqui você aprenderá sobre todas as mecânicas do Armageddon MC. Das mecânicas de sistemas mais simples até as mais complexas. Lista com todos os comandos que ajudam na gameplay!

Passo-a-passo com informações valiosas para tornar sua experiência no Armageddon MC ainda melhor. Abaixo temos a lista dos tutoriais prontos, muitos outros serão adicionados com o tempo!

## Primeiros Passos no Servidor

### 🏴‍☠️ 1. Leia as Regras!

Todas as regras importantes estão em: https://wiki.armamc.com/regras

### 🧭 2. Escolha o Servidor!

Ao entrar no lobby do Armageddon, clique com a bússola na mão e escolha um dos dois servidores:

* Survival Gênesis - Aberto em 23/06/2019
* Survival Apocalipse - Aberto em 04/04/2020

### 🗺️ 3. Explore o Mapa!

Use o comando **`/wild`** e clique em confirmar para ser teleportado para um local aleatório do mapa! Cada teleporte aleatório possui um tempo de espera de 30 minutos para ser utilizado novamente.

Você também pode sair andando, o **Spawn** possui uma área de 500x500.

### 🏳️ 4. Defina Sua Base!

Use o comando **`/sethome [nome]`** para definir sua base, você poderá voltar para este ponto sempre que quiser com o comando **`/home [nome]`**.

Exemplo: **`/sethome base`** para definir e **`/home base`** para voltar ao local.

### 💂 5. Proteja sua Base!

Tutorial de como proteger suas construções:

* Use **`/claim`** para proteger ao seu redor.
* Use **`/prot`** para gerenciar sua proteção.

### 💬6. Envie mensagens!

Se comunique com outros jogadores:

* Use **`/g mensagem`** para falar no chat global;
* Use **`/m <jogador> <mensagem>`** e converse no chat privado com outro jogador;

* Use **`/mail send <nick> <mensagem>`** para enviar um e-mail para jogadores offline.

### 💰 7. Compre e Venda!

Use o comando **`/loja`** para abrir o menu de lojas. Você pode ganhar dinheiro dentro do servidor de várias formas, vender itens é apenas uma delas.
